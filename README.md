## 实验内容概览

### 实验1：第一个微信小程序
- 学习使用快速启动模板创建小程序
- 学习手动创建小程序的方法
- 掌握小程序基本结构和视图设计

### 实验2：微信小程序基础组件
- 学习常用组件（图片、文本、按钮等）
- 掌握组件的事件绑定
- 实现简单的交互功能

### 实验3：微信小程序API应用
- 学习微信小程序 API 的使用
- 实现多媒体功能
- 集成第三方数据

### 实验4：页面跳转与自定义组件
- 掌握页面间的跳转和数据传递
- 学习自定义组件的开发
- 实现组件化开发

### 实验5：表单与数据绑定
- 学习表单组件的使用
- 掌握数据绑定和双向绑定
- 实现用户输入处理

### 实验6：推箱子游戏
- 综合运用所学知识开发完整游戏
- 实现游戏逻辑和界面设计
- 自定义组件开发（倒计时组件）

### 期末项目：海大拼车
- 完整的拼车平台应用
- 云开发集成（数据库、云函数、云存储）
- 实时拼车功能
- 用户系统和订单管理

## 环境要求

- **开发工具**: [微信开发者工具](https://developers.weixin.qq.com/miniprogram/dev/devtools/download.html)
- **运行环境**: 微信客户端 6.7.2 及以上版本
- **开发语言**: JavaScript、WXML、WXSS、JSON
- **云开发**: 微信云开发（用于期末项目）

## 快速开始

### 安装开发工具

1. 下载并安装 [微信开发者工具](https://developers.weixin.qq.com/miniprogram/dev/devtools/download.html)
2. 使用微信扫码登录
3. 选择对应的项目文件夹打开

### 导入项目

1. 打开微信开发者工具
2. 点击"导入项目"或"小程序"
3. 选择对应实验或项目的目录
4. 填写 AppID（可使用测试号）
5. 点击"导入"

### 项目配置

每个项目的 `project.config.json` 包含项目配置信息，包括：
- 项目名称
- AppID
- 项目配置
- 云开发环境 ID（finalProject）

## 开发规范

### 项目命名

- 实验项目：`Lab-N`（N为实验编号）
- 微信小程序项目：`MP`、`MP1`、`MP2` 等
- 云函数目录：`cloudfunctions`
- 小程序源码目录：`miniprogram`

### 代码规范

- 使用 ESLint 进行代码检查（部分项目包含 `.eslintrc.js`）
- 遵循微信小程序官方代码规范
- 组件和页面命名采用小驼峰命名法

### 文档规范

- 实验报告使用 Markdown 格式
- 代码块标注语言类型
- 图片使用相对路径引用

## 云开发说明

期末项目使用微信云开发，包含三大基础能力：

1. **数据库**: JSON 文档型数据库，用于存储用户信息、订单数据等
2. **文件存储**: 云存储功能，用于存储用户头像、车辆图片等
3. **云函数**: 云端运行的代码，处理业务逻辑

详细说明请参考：[云开发文档](https://developers.weixin.qq.com/miniprogram/dev/wxcloud/basis/getting-started.html)
