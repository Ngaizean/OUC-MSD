# 微信小程序实战项目集

> 从入门到实战的完整微信小程序开发项目集合

## 项目简介

本仓库包含一系列微信小程序实战项目，涵盖从基础入门到完整商业应用的开发全过程。项目采用渐进式学习路线，通过多个实际项目案例，帮助开发者快速掌握微信小程序开发技能。

主要项目包括：天气应用、垃圾分类工具、媒体播放器、新闻客户端、推箱子游戏以及完整的拼车平台应用。

## 技术栈

- **开发平台**: 微信小程序
- **前端框架**: 原生小程序框架 (WXML/WXSS/JavaScript)
- **后端服务**: 微信云开发（云数据库、云函数、云存储）
- **UI组件**: 自定义组件 + WeUI
- **开发工具**: 微信开发者工具

## 快速开始

### 环境准备

1. 下载并安装 [微信开发者工具](https://developers.weixin.qq.com/miniprogram/dev/devtools/download.html)
2. 注册微信小程序并获取 AppID（也可使用测试号）
3. 对于云开发项目，需要开通云开发服务

### 项目导入

1. 打开微信开发者工具
2. 选择"导入项目"或"小程序"
3. 选择对应项目的目录
4. 填写 AppID（可使用测试号）
5. 点击"导入"

### 云开发配置

使用 `finalProject`、`Lab-3` 等包含云开发功能的项目时：

1. 在微信开发者工具中点击"云开发"按钮
2. 创建云开发环境
3. 在 `project.config.json` 中配置云环境 ID
4. 上传云函数：右键 `cloudfunctions` 目录下的云函数文件夹，选择"上传并部署：云端安装依赖"
5. 创建数据库集合并设置权限

## 开发规范

### 命名规范
- 项目目录：使用有意义的英文名称
- 组件和页面：采用小驼峰命名法
- 文件名：使用小写字母和连字符

### 代码规范
- 使用 ESLint 进行代码检查
- 遵循微信小程序官方代码风格
- 保持代码简洁、可读
- 添加必要的注释说明

### 文档规范
- 项目文档使用 Markdown 格式
- 代码块标注语言类型
- 图片使用相对路径引用

## 常见问题

**Q: 云开发如何收费？**

**Q: 可以用于商业项目吗？**
A: 可以，但建议根据实际需求进行优化和扩展。
