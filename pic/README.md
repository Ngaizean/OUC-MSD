# 实验报告截图

本目录用于存放《移动软件开发》课程的实验报告截图，对应各个实验的运行结果和关键步骤截图。

## 目录结构

```
pic/
├── Lab-1/    # 实验1：第一个微信小程序
├── Lab-2/    # 实验2：简易的天气小程序
├── Lab-3/    # 实验3：垃圾分类小程序
├── Lab-4/    # 实验4：媒体API之口述校史
├── Lab-5/    # 实验5：高校新闻网
└── Lab-6/    # 实验6：推箱子游戏
```

## 图片说明

各实验目录下的图片按顺序编号（1.jpg, 2.jpg, ...），在对应实验的 README.md 中通过以下方式引用：

```markdown
<img src="../pic/Lab-X/1.jpg" alt="步骤1" style="zoom:33%;" />
```

其中 `Lab-X` 为实验编号，`1.jpg` 为图片文件名。

## 注意事项

- 图片格式为 JPG
- 图片按实验步骤顺序编号
- 图片在 README 中引用时使用相对路径
- 如需添加新截图，请保持命名规范一致
